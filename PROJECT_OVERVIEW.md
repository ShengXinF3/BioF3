# BioF3 项目总览

## 项目定位

BioF3 是一个基于 Docusaurus 的中文生物组学教程站点。

## 当前保留内容

- `docs/`：教程正文
- `blog/`：博客文章
- `src/`：页面、组件和样式
- `static/`：图片和静态资源
- `scripts/`：构建与部署脚本
- `README.md`：项目入口说明
- `REMOTE_DEPLOY_GUIDE.md`：远程服务器发布说明

## 发布方式

当前项目支持：

- 本地开发：`npm start`
- 构建：`npm run build`
- 本地预览：`npm run serve`
- 远程同步：`bash scripts/deploy-biof3.sh`

## 维护原则

- 教程正文和博客内容保留
- 过程性总结、一次性修复记录、临时调试文档删除
- 新的长期说明文档只保留一份，避免重复堆积

## 备注

如果后续再新增文档，优先判断它是否属于以下三类之一：

1. 读者长期需要的入口文档
2. 内容作者长期需要的维护文档
3. 只属于某次修复/任务的过程记录

第 3 类默认不保留。
