# bulk RNA-seq 实践教程

本专栏规划 bulk RNA-seq 数据分析的完整实践流程，后续会逐步补充真实数据、可运行脚本和结果解释。

## 实践路线

1. 数据集选择与下载
2. FASTQ 质量控制
3. 比对或准比对
4. 表达量定量
5. 样本质控与批次检查
6. 差异表达分析
7. 功能富集分析
8. 结果可视化与报告整理

## 计划模块

- Module 01：实践数据集与项目设计
- Module 02：FASTQ 质量控制与 MultiQC
- Module 03：STAR / HISAT2 比对流程
- Module 04：Salmon / kallisto 定量流程
- Module 05：DESeq2 差异表达分析
- Module 06：GO / KEGG / GSEA 富集分析
- Module 07：火山图、热图和表达趋势图
- Module 08：可复现报告与结果交付
