# 基因组学实践教程

本专栏规划基因组测序数据分析的实践流程，重点覆盖从原始测序数据到变异结果解释的关键步骤。

## 实践路线

1. 数据集选择与参考基因组准备
2. FASTQ 质量控制
3. 读段比对与 BAM 文件处理
4. 变异检测
5. 变异过滤与质量评估
6. 变异注释
7. 群体遗传和可视化分析
8. 结果整理与可复现交付

## 计划模块

- Module 01：实践数据集与参考资源
- Module 02：BWA / minimap2 比对
- Module 03：SAMtools 与 BAM 文件处理
- Module 04：GATK 变异检测流程
- Module 05：VCF 过滤与质量控制
- Module 06：ANNOVAR / VEP 变异注释
- Module 07：群体结构和 PCA 分析
- Module 08：结果报告和复现清单
