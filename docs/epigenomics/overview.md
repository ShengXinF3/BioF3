# 表观组学实践教程

本专栏规划表观组学数据分析教程，覆盖染色质可及性、蛋白-DNA 结合和 DNA 甲基化等方向。

## 实践路线

1. 数据集选择与实验类型识别
2. FASTQ 质量控制
3. 比对与去重复
4. peak calling
5. peak 注释与 motif 分析
6. 差异 peak 分析
7. 与转录组结果整合
8. 结果可视化与报告

## 计划模块

- Module 01：实践数据集与实验设计
- Module 02：ATAC-seq 标准分析流程
- Module 03：ChIP-seq 标准分析流程
- Module 04：peak calling 与质量评估
- Module 05：peak 注释和 motif 富集
- Module 06：差异开放区域分析
- Module 07：甲基化数据分析入门
- Module 08：表观组与转录组整合
